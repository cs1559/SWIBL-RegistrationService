
SWIBL-GameService
===============================================================================================

The SWIBL-GameService project is a micro-service implementation to support GAME management
for the overall league/baseball management application.  The service will  support the following
functions:

- retrieve an individual game
- retrieve a team game schedule for a given season

This service is expected to be deployed within a folder (games) under the root note.

POST - create
GET - read
PUT - update/replace
PATCH - Update/modify
DELETE - Delete
